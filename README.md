# 献立アプリ

⚠️ このプロジェクトは現在開発中です。
フロントエンド・バックエンド・データベースを使った Web アプリの練習として作成中です。

## 概要

毎日の献立（朝・昼・夜）を登録・参照できるアプリです。
最低限の機能で実装することを目的としています。

## 制作の目的

- データベースと Web アプリとの連携を学ぶ
- HTML・DOM 操作を用いた UI でシンプルに作成する

## 使用技術(予定)

（※ 開発中のため変更の可能性があります）

- Frontend: TypeScript, Vite, HTML
- Backend: Node.js (Express)
- Database: PostgreSQL
